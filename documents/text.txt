// Steepest-Ascent Hill climbing : It first examines all the neighboring nodes and then selects the node closest to the solution state as of next node.   
// Stochastic hill climbing : It does not examine all the neighboring nodes before deciding which node to select .It just selects a neighboring node at random and decides
